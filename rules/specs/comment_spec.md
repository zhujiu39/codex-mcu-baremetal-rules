# 注释规范详情

> 触发条件：任何新增或修改 `.c`、`.h` 文件、函数注释、模块说明、接口说明、重要逻辑说明的任务，必须先阅读本文件。
> 此文件为按需加载的详细参考，核心规则见 `AGENTS.md`。

## 语言规则

| 场景 | 语言 | 说明 |
|------|------|------|
| 解释与交互 | 简体中文 | 最高优先级 |
| 代码注释（非 main.c） | 简体中文 | 保持简洁，避免特殊符号 |
| `main.c` 中的注释 | 英文 | CubeMX 兼容性要求 |

`main.c` 是唯一注释语言例外。除 `main.c` 外，项目自定义模块注释默认使用简体中文。

## 模块级文件头注释

每个自定义 `.c` 文件顶部必须写模块说明，放在 `#include` 前。`main.c` 不强制补文件头，避免干扰 CubeMX 生成结构。

```c
/**
 * @file    dht11.c
 * @brief   DHT11 温湿度传感器驱动，包含 EMA 滤波
 * @deps    HAL_GPIO, HAL_TIM
 */
```

字段说明：

* `@file`：当前文件名。
* `@brief`：模块职责，不写空泛描述。
* `@deps`：依赖的 HAL 外设、驱动模块或硬件资源。

## 函数注释模板

函数注释保留必要信息，避免过重模板导致形式主义。

必填字段：

* `@功能`
* `@日期`
* `@参数`
* `@返回值`
* `@使用说明`

按需字段：

* `@引脚连接`
* `@前置条件`
* `@调用频率`
* `@硬件依赖`
* `@线程安全`
* `@变更`

模板：

```c
/**
 ******************************************************************************
  @功能：[一句话描述函数核心功能]
  @日期：[YYYY-MM-DD]
  @参数：[输入] param1 - 参数说明
         [输出] param2 - 指针返回值说明
         无参数时写：无
  @返回值：[类型] - 含义，void 时写：无
  @使用说明：[调用场景、频率限制、失败处理]
 ******************************************************************************
 */
```

## 重大逻辑变更记录

`@变更` 只用于重大逻辑变化，不要求每次小修都写。

```c
/**
 ******************************************************************************
  @功能：解析 ESP01S 下发的控制帧
  @日期：2026-05-11
  @参数：[输入] line - 已完成换行分帧的 JSON 行，保证以 '\0' 结尾且不含 '\n'
  @返回值：ErrCode - 解析结果
  @使用说明：仅处理完整帧，不负责串口接收；字段范围校验由调用方继续完成
  @变更：2026-05-11 将串口协议调整为 JSON Lines
 ******************************************************************************
 */
```

## main.c 英文注释示例

`main.c` 中注释必须使用英文字段名和英文说明。

```c
/**
 ******************************************************************************
  @brief  Initialize application modules
  @date   2026-05-11
  @param  None
  @retval None
  @note   Must be called after MX_xxx_Init() functions
 ******************************************************************************
 */
```

## 行内注释规范

正确注释应解释设计意图、硬件约束或边界条件。

```c
#define DS18B20_CONV_MS 750U  /* DS18B20 12-bit conversion time */
```

避免重复代码本身含义。

```c
heater_on();  /* 错误示例：打开加热片 */
```

函数名已经清楚表达行为时，不需要行内注释。
